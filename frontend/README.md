# Frontend

Next.js (App Router) frontend for Project Management Studio, exported to a static site and served by the backend. See the root [README](../README.md) for the full app.

## Develop

```bash
npm install
npm run dev
```

## Test

```bash
npm run test:unit
npm run test:e2e
```

## Build

```bash
npm run build
```

Produces a static export in `out/`.
